CARACTERISTICAS:

  1. Capacidad para múltiples programas.

  2. Aplicable a diferentes hardwares.

  3. Concentración de las funciones más basicas de la computadora.

  4. Comunicación con el usuario a través de una interfaz gráfica.

¿QUÉ ES UN SISTEMA OPERATIVO?

Un sistema operativo es un conjunto de programas que controla el funcionamiento de las partes fisicas de un equipo electrónico, facilitando al usuario el uso de la computadora u otro equipo. Es decir es el software que maneja el hardware.

FUNCIONES:

El sistema operativo administra los recursos ofrecidos por el hardware y actúa como un intermediario entre la computadora y su usuario. Además proporciona un ambiente en donde el usuario pueda ejecutar programas en una forma convenientes y eficiente.

MS/DOS

Fue diseñado por microsoft para las computadoras personales IBM en 1981. Podía administras discos, archivos y dispositivos de entrada y salida.

MICROSOFT WINDOWS

Ampliamente utilizado en las computadoras personales. Es una familia de sistemas operativos que ha evolucionado.

LINUX

Sistema operativo de dominio público y gratuito. El usuario puede selecionar el administrador de ventanas de su preferencia.
